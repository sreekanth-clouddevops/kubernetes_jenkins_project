About this demo

This is a minimal static site to verify that Apache (httpd) inside the container is serving files correctly. Replace this content with your real app files (HTML, CSS, JS).

Files served

Place your files in the Apache document root:

/usr/local/apache2/htdocs/

The default file served by Apache is index.html.

Quick health checks

  • Visit http://<server-ip>/ in the browser — you should see this page.
  • From inside the container: curl -I http://localhost/ should return HTTP/1.1 200 OK.
  • A simple file test: ls -la /usr/local/apache2/htdocs/

Example curl test

curl -sS http://localhost/ | head -n 20
